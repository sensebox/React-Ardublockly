# Add Class

A **class** represents one orientation you want to recognise — for example "flat", "upright left", or "tilted 45°".

## Tips

- Use **descriptive names** that remind you of the physical position (e.g. "screen up", "screen down").
- You need **at least 2 classes** to train a model.
- You can add up to **5 classes** per model.
- Click the class name to rename it at any time.
