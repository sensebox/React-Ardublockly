Think of the model as a complex mathematical function. Before the model is trained it still has many **unknown values**, which are given random values initially.

![Math Neural Network](/media/teachable/math.png)

During training, the model is shown the training data and tries to classify it. If it does not classify well, it adjusts its values a little.

![training process](/media/teachable/training.png)

This process is repeated many times. In this way, the individual neurons gradually learn certain **patterns** in the input and together form a model that can make useful predictions.

![trained](/media/teachable/trained.png)
